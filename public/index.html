User Stories

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example to list recent 10 search results

https://rk-image-search-abstraction.herokuapp.com/api/latest/imagesearch

Example to do an image search

https://rk-image-search-abstraction.herokuapp.com/api/imagesearch/<searchstring>

Use ?offset=10 for next page results
